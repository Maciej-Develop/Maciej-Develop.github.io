# Daihiti

- Position: Full Stack Developer Intern
- Period: 09/2025 - 01/2026

## Description

During my internship at Daihiti, I contributed to the continued development of the web platform TimeWoop, a website that allows users to schedule messages and attach gifts to them.

I worked closely with a small development team to implement new features, improve existing functionality, and maintain the platform.

This experience helped me strengthen my full-stack development skills and gain valuable insight into working in a startup environment.

## Key Contributions

- Developed a feature allowing users to generate images via an API and attach them to scheduled messages. Implemented the frontend interface with VueJS and integrated backend services in Laravel, including AWS storage and external image generation APIs.

- Built a mobile version of the platform using CapacitorJS to convert the web application into a mobile app.

- Performed quality control and testing to ensure existing platform features functioned correctly.

- Developed multiple Vue.js components for the Pro version of the platform, improving the user interface and functionality.
  
## Tech Stack

Laravel, VueJS, Git, AWS S3, CapacitorJS
