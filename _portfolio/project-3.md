# FrameFlow

FrameFlow is an Android mobile application that allows users to discover new movies and shows, search for them, and manage their favourites as well as the ones they have already watched.

The application uses [The Movie Database API](https://www.themoviedb.org/) and Firebase for storage and authentication.

[Git repository here](https://github.com/Maciej-Develop/FrameFlow)

## Features

- Discover trending movies and shows  
- Search for any movie or show  
- Add movies/shows to favourites and to a backlog  
- Authentication via a Google account
